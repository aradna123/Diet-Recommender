# 🥗 Personalized Diet Recommender System

## 📌 Overview

This project is a **personalized diet recommendation system** that generates nutrition plans based on a user’s health conditions and disease severity levels. By combining **frontend design, Django-based backend APIs, and a MySQL database**, the system delivers dynamic, condition-specific recommendations in real-time.

## 🔬 Research Motivation

The project demonstrates how computational frameworks can support **health-focused decision-making**. While the current system uses structured rules for diet mapping, it opens pathways for **future research**, including:

* Using **machine learning models** to predict optimal diets from health datasets.
* Exploring **semantic search / NLP** to connect medical knowledge with personalized recommendations.
* Investigating the **ethical and safety challenges** in automated health guidance.

## ⚙️ Tech Stack

* **Frontend**: React
* **Backend**: Django REST Framework
* **Database**: MySQL
* **API Integration**: Django APIs for CRUD and recommendation logic

## ✨ Key Features

* User input for health and disease condition levels.
* Dynamic generation of diet plans tailored to condition severity.
* Secure backend integration with **Django REST APIs**.
* Persistent storage of health data and diet history in **MySQL**.
